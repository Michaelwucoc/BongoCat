## Background

This project is inspired by [MMmmmoko](https://github.com/MMmmmoko)'s [Bongo-Cat-Mver](https://github.com/MMmmmoko/Bongo-Cat-Mver), which is loved for its unique cat interaction features but only supports Windows. As a macOS user, I wanted to use this cute cat on my own device, so I decided to develop a macOS-compatible version.

Thanks to [Tauri](https://github.com/tauri-apps/tauri)'s powerful cross-platform capabilities, this project not only supports macOS, but also Windows and Linux(x11), allowing more users to interact with this adorable cat!

## Features

- Supports macOS, Windows, and Linux(x11).
- Cat moves or taps keyboard in sync with your keyboard or mouse actions.
- Supports importing custom models to create your own cat.
- Fully open source, transparent code, no user data collection.
- Works offline, no network required, privacy protected.
